Rearrange

============

This module is used for rearranging names.
Turns "Last_name First_name" format to "First_name Last_name" format.

## Example

 * Calling rearrange_name("Turing", "Alan") to "Alan Turing"
 * Calling rearrange_name("Hopper", "Grace M.") to "Grace M. Hopper"
 * Calling rearrange_name("Voltaire") to "Voltaire"
